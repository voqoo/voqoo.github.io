Algorithm

Divide the n-element sequence to be sorted into two subsequences of n/2 elements each.

Conquer: Sort the two subsequences recursively using merge sort.

Combine the two sorted subsequences to produce the sorted answer.Merge by calling the auxiliary procedure MERGE(A,p,q,r),where A is an array and p,q,and r are indices into the array to describe the subarrays.
