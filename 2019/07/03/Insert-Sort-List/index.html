Insert Sort List

2019-07-03

mathjax: true

Problem

Input:

A sequence of n numbers

​ $$

Output:

A permutation(reordering) of the input sequence

​ $<a_1^{‘},a_2^{‘},…,a_n^{‘}>$

such that

​ $a_1^{‘}\leq a_2^{‘}\leq …\leq a_n^{‘}$

Algorithm

Simulate the process of insert sort with list.

Use 3 arrays to store the list.

bb[i] means “bigger brother”,the left node of the current node.

lb[i] means “little brother”,the right node of the current node.

Everytime we get a new element we insert it into the list by traversing the list to find the first node whose key is bigger than the element and place it on the left.
